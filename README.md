# Simple CRUD Server

A RESTful backend API built with Express.js and MongoDB to manage user data efficiently. This server provides complete CRUD (Create, Read, Update, Delete) operations and serves as the backend for the Simple CRUD application.

## Technologies Used

- Node.js
- Express.js
- MongoDB
- dotenv
- cors

## Core Features

- Retrieve all users from the database
- Fetch individual user details by ID
- Create and store new users
- Update existing user information
- Delete users from the database
- REST API architecture
- Environment variable security using dotenv
